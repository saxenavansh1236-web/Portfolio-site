  • Role-based admin panel (Admin/Analyst/Viewer) with case management, scheduled monitoring, SMTP/webhook alerting, audit logs, and CSV/PDF/JSON exports
  • Four-layer defense-in-depth security: Flask-Limiter rate limiting, custom SQL injection guard, HMAC-signed math CAPTCHA, and JWT access/refresh tokens for the API surface
